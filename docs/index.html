토큰 재발급 가이드

토큰 오류 시 아래 순서로 재발급하세요. Refresh Token은 약 2개월 유효하며, 자동 갱신되지만 장기 미사용 시 만료됩니다.

1
카카오 개발자 콘솔에서 앱 확인 → REST API 키 복사
2
카카오 로그인 → Redirect URI에 http://localhost:3000 등록 확인
3
동의항목 → talk_message 활성화 확인
4
로컬에서 python setup_kakao.py 실행 → 브라우저 인증 → 코드 입력
5
발급된 토큰을 GitHub Secrets에 등록:
KAKAO_REST_API_KEY — REST API 키
KAKAO_CLIENT_SECRET — 보안 키 (선택)
KAKAO_REFRESH_TOKEN — Refresh Token
6
워크플로우를 수동 실행하여 정상 동작 확인
